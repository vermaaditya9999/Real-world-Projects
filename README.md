# Expense Tracker (Python CLI)

A command-line based Expense Tracker developed in Python that enables users to efficiently record, manage, and analyze their daily expenses using persistent file storage.

## Features

- Add expense (date, amount, description, category)
- View all expenses
- View total expenses
- Search by category
- Filter by date range with total
- Budget / overspending check
- Exit program

---

## Tech Used

- Python 3
- File Handling
- datetime module
